### Instalação de dependências
#### SDL2 

Com um gerenciador de pacotes em sistemas `Linux/Unix`:
```bash
sudo apt-get install libsdl2-dev
```
#### SDL2_TTF 

Com um gerenciador de pacotes em sistemas `Linux/Unix`:
```bash
sudo apt-get install libsdl2-ttf-dev
```
### Compilar e Rodar
    fpc -Fu./lib/ main.pas
    ./main
